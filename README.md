# ✊ Rock, 🤚 Paper, ✌️ Scissors

A simple browser-based implementation of the classic Rock, Paper, Scissors game. This project is built using JavaScript and features a dynamic user interface.

## 🚀 Live Demo
[View the Live Preview here](https://felippemenezes.github.io/rock-paper-scissors/)

## Features

- **Interactive UI**: Uses buttons for player input and features a smooth randomization animation for the computer's choice.
- **Dark/Light Mode**: Includes a theme toggle to switch between dark and light visual styles seamlessly.
- **Computer AI**: The computer makes a random choice between Rock, Paper, and Scissors using a randomized selection logic.
- **Round History**: Keeps track of every round's outcome in a scrollable list for easy review.
- **Scoring System**: Tracks scores for both the human player and the computer.
- **First to 5 Wins**: The game continues until either the player or the computer reaches 5 points, announcing a final winner.
- **On-screen Feedback**: Provides real-time updates on round results, scores, and game status directly on the page.

## How to Play

1. Clone this repository to your local machine.
2. Open the `index.html` file in any modern web browser.
3. Click on the **Rock**, **Paper**, or **Scissors** buttons to make your move.
4. Click the **PLAY** button to trigger the computer's choice and see the result.
5. Reach 5 wins to claim victory! You can restart the game once it's over.

## Technical Overview

The project demonstrates fundamental JavaScript concepts:
- Variable scoping and constants.
- Functions and return values.
- Conditional logic (`if/else`, `switch`).
- DOM Manipulation: Creating elements, updating text content, and style injection.
- Event Listeners: Handling user interactions through click events.
- CSS Variables: Implementation of dynamic themes using custom properties.
- Asynchronous JavaScript: Using `setTimeout` and callbacks to manage game animations.
