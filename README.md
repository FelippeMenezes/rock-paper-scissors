# rock-paper-scissors

A simple browser-based implementation of the classic Rock, Paper, Scissors game. This project is built using JavaScript and is played through the browser's console and prompt windows.

## 🚀 Live Demo
[View the Live Preview here](https://felippemenezes.github.io/recipes-top/)

## Features

- **Interactive Gameplay**: Uses `prompt()` to receive user input directly in the browser.
- **Computer AI**: The computer makes a random choice between Rock, Paper, and Scissors using a randomized selection logic.
- **Scoring System**: Tracks scores for both the human player and the computer.
- **Best-of-5 Format**: The game continues until either the player or the computer reaches 3 wins.
- **Console Feedback**: Provides real-time feedback on round results and current scores via the browser console.

## How to Play

1. Clone this repository to your local machine.
2. Open the `index.html` file in any modern web browser.
3. Open the **Developer Tools** (usually `F12` or `Right-click > Inspect`).
4. Navigate to the **Console** tab.
5. Follow the browser prompts to enter your choice: `rock`, `paper`, or `scissors`.
6. Check the console to see who won the round and watch the score climb!

## Technical Overview

The project demonstrates fundamental JavaScript concepts:
- Variable scoping and constants.
- Functions and return values.
- Conditional logic (`if/else`, `switch`).
- Loops (`while`) for game state management.
- Basic DOM interaction via `prompt()`.
